# 📡 Tech Slop Bot

Личный ИИ-агент для мониторинга ИТ-новостей из Telegram-каналов. Парсит источники, стандартизирует тексты через LLM и доставляет готовую ленту в личные сообщения.

## Что умеет

- Мониторит список Telegram-каналов и подхватывает новые посты
- Приводит каждую новость к единому редакционному формату через LLM
- Классифицирует новости по тематическим категориям
- Управляется командами в боте: `/add`, `/remove`, `/list`

## Как работает

При каждом запуске бот проверяет все добавленные каналы через `t.me/s/channel`, находит новые посты и прогоняет каждый через MiMo-V2.5-Pro — LLM стандартизирует текст, извлекает заголовок, детали и категорию. Готовые новости отправляются в Telegram. Состояние сохраняется в JSON-файлах в репозитории.

Запускается на GitHub Actions через cron-job.org (тестовый режим).

## Формат новости

```
<Жирный заголовок с гиперссылкой на источник.>

#хэштег_категории

```
